The display property has values of inline block inline-block flex grid etc.

inline elements actually have a width of the content which the contain. Margin top and bottom is not really applied to them
To add a margin top or margin bottom, you have to set the display property to
inline-block

Block level elements on the other hand occupy the full width of the screen except if the width is explicitly defined.
If the width is explicitly defined, and you would like to center the element, you can type:
margin: 0(or any value of your choice for top and bottom) auto(for the left and right)
The above code will center it

display: none;This will completely remove the element from the flow of the document.
opacity: 0;This will hide the element or make the element transparent but the element will still be in the flow of the document visibility: hidden; Almost the same as opacity: 0;
