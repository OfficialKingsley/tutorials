This is actually used for styling the page
As we know, the html of a page is used to structure the page

Syntax

These are called the css rules

selector {
    property: value;
    property: value;
}

It should be noted that the internal stylesheet will override an external stylesheet while the inline style will override the internal stylesheet

property: value; is called a declaration.
{property: value; property: value2;} is called a declaration block
h1, h2, .class #id are all selectors
h1 {property: value;} is then finally called a css rule
