CSS units are divided into:
  • Absolute units
  • Relative units e.g. %, em, rem, vh (view-height: height of the screen), vw(view-width: width of the screen)

Pixel

Pixel is an absolute unit where 1 pixel represents approximately 1 dot on the value
The reason pixel is an absolute value is that no matter the screen size, it does not change. It is always constant 

h1 {
    font-size: 60px;
    width: 300px;
    height: 400px;
    background: blue;
}

% value

.div1 {
    width: 600px;
}
.div2 {
    width: 50%;
}
/* The width of div2 is going to be 50% relative to the parent element*/

calc() function in css

This helps us to calculate values min-height: calc(100vh - 100px)

Properties of height include height, min-height
