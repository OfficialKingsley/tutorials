Everything in html is actually represented in a box

margin: This is the gap between the border and the screen or another element

Padding: This is the distance between the content and the border of the element

padding-top: ;
padding-bottom: ;
padding-right: ;
padding-left: ;
padding: value for all sides;
padding: (value for top and bottom) (value for left and right);
padding: top right bottom left;

                
margin-top: ;
margin-bottom: ;
margin-right: ;
margin-left: ;
margin: value for all sides;
margin: (value for top and bottom) (value for left and right);
margin: top right bottom left;

The margin can also have negative values which will shift it to the opposite direction 

border-style: ; solid, dotted, double
border-color: ;
border-width: ;
border: width style color;
border-bottom-style: ;
border-bottom: width style color;
border-radius: ; This rounds off the border
                
outline-style: ; solid, dotted, double
outline-color: ;
outline-width: ;
outline: width style color;
outline-bottom-style: ;
outline-bottom: width style color;
outline-radius: ; This rounds off the border
outline-offset: ; Distance between the outline and the borders of the element
