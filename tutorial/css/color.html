The color property changes the color of text and can have values in different forms

Color names, rgb, rgba, hex, hsl

background-color

rgb stands for red green and blue with each value (red greed or blue) having values ranging from 0 to 255
color: rgb(255, 0, 0) Will give red color: rgb(0, 2555, 0) Will give green color: rgb(0, 0, 255) Will give blue

rgba stands for rgb with opacity
By default, its value is one
rgba(0, 0, 0, 0.4)
The value of the "a" value ranges from 0 to 1. This value represents the opacity of the color

hex values. These values are in hexadecimal value
#ff0000
#f26a7

Each value has a value ranging from 0 - f

#f00 = #ff0000

You can also have opacity values in hexadecimal values.
In this case, you'll need to have four or 8 values 

h1 {
    color: #ff2a8eac;
    background-color: #a2c4
}
