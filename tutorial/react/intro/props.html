Props

Since we are using functions, the props are the parameters of the function components 

const Header = (props) => {
    return {
        <header>
            <h1>{props.title}</h1>
        </header>
    }
}
export default Header //Notice that we exported it

depending on where it is imported, you can do this

import Header from "Header"
<Header title="my Title" />

What the above will do is that it will display my Title which was defined as props.title in the h1 element.

You can also give default values to the props since the props is an object. This can be done as follows 

Header.defaultProps = {
    title: 'Task Tracker'
}

//This should be done in the Header.js

So if the props.title is not given, then the default is used

Since the props is an object, we can destructure and the above can be done as follows:

const Header = ({ title }) => { 
  //See that we used an object
  //In better terms, we destructured the props object
    return {
        <header>
            <h1>{title}</h1> //We removed props
        </header>
    }
}
export default Header

Prop types

import PropTypes from 'prop-types'

Header.propTypes = {
    title: PropTypes.string
}
                
import PropTypes from 'prop-types'

//If it is required,
Header.propTypes = {
    title: PropTypes.string.isRequired
}

So if we do this

<Header title={2} /> //Notice the use of curly braces

//We use curly braces when the prop is not a string

An error will be generated

Note that when putting in props if they are not strings, we use curly braces
