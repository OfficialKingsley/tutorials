When you map through an array using the .map, and you output it in a JSX, you create a list. Each item in the list must have a unique key prop. 

tasks.map((task)=> (
    <h3 key={task.id}>{task.text}</h3>
))

//Notice that we used the normal brackets instead of the curly braces
//This is called an instant return since we are returning jsx instantly

Using states

In functional based components, to use state, we import the useState hook from react 

import { useState } from 'react'
//Then in the component,

const Tasks = () => {
    const [tasks, setTasks] = useState(
      [
        {
            id:
            text:
            day:
            reminder:
        }
        {
            id:
            text:
            day:
            reminder:
        }
        {
            id:
            text:
            day:
            reminder:
        }
        {
            id:
            text:
            day:
            reminder:
        }
      ]
    )
}

//Note that states are immutable. So if we type something like tasks.push(), we will most likely get an error. You basically recreate it and send it. It is called a one way data

setTasks ([New array of values]) //This will create a new array of values
setTasks ([...tasks, {}])
