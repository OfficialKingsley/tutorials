Styling

There are different ways. You can use the style sheet index.css you can also use something like styled components.

You can also use inline like this

<h1 style= {{color: 'red', backgroundColor: 'black'}}>Hello this is my style</h1>

//Note the double curly braces
//And also notice that we use camel case naming syntax

Another thing you could do is to do this:

<h1 style = {headerStyle}>Hello my Styled title </h1>

//A variable containing the styles
const headerStyle = {
    color: 'red',
    backgroundColor: 'black'
}
