useRef hook

This hook is the easiest way to access dom elements
This hook is used when you want to focus on an input whenever an action is taken.
Normally is react, we do not have to access dom elements because of the virtual dom. However, there are some situations where you would need to manipulate or at least add some functionality to certain dom elements and the useRef helps in that

To use the useRef hook, the element that we want to access must have a ref attribute 

import {useRef} from "react";

const RefTutorial = () => {
    const inputRef = useRef(null);

    const clickEvent = () => {
        console.log(inputRef.current.value); //Notice that we had to access the current value
    }

    return (
        <div>
            <h1>Pedro</h1>
            <input type="text" placeholder="Some random text here ref={inputRef}/> 
            //You pass in the name of the created ref created using the useRef hook
            <button onClick={clickEvent}>Change Name<button>
        </div>
    );
}
