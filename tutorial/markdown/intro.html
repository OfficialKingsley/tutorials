So, what is markdown?
It is a lightweight markup language with plaintext formatting syntax i.e. instead of using tags, it uses just some simple text formatting
An amazing feature to the markdown is that it can be converted to html or xhtml and other formats
One of the main purposes is for readability and easiness to use

Uses

  • Readme Files (Github etc.)
  • Forum posts and blog posts
  • Used also in many static site generators

Basic types of formatting

  • Headings
  • Lists
  • Emphasis (bold)
  • Links
  • Blocks of code
  • Images
  • Blockquotes
  • Horizontal rules

Where to write markdown

  • Text Editors Extensions (VSCode, Atom)
  • MarkPad
  • HarooPad
  • MarkdownPad 2
  • Typora
