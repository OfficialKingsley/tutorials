Using a database

In mongo db, you can use a db even if it is not created. It will automatically create the database for you when you input data into it

Also note that in Mongo DB, we have collections(like the tables in the database) and we have databases(like in sql)

To use a db 
use dbname

To show collections, we do this

show collections

If you are in a database and you want to delete the database, you can just type
db.dropDatabase() db.dropCollection() to drop the collection like a Javascript code
However, if we delete a database, we will still appear to be in that database but if you type show dbs, you'll see that the database is no longer in the list even though we appear to be in that database

To get your current database, you type db
