In local storage, items are stored in key value pairs

The local storage is a storage object with some methods and properties

localStorage.setItem(key,  value)
//The above is the same as
window.localStorage.setItem(key, value)
                
localStorage.setItem("name", "myName")

Even if the code is deleted and the page is refreshed the item you set will still be in the local storage

Removing and item

This can be done by using the the .removeItem(key) method

Getting an item from the local storage

This can be done using the .getItem(key) method

It should be noted that the local storage automatically stringifies the values of the key-value pairs

Getting the name of a key

This can be done using the .key(index) method

This index is the index of the key you wanna get
