# Arrow functions

These are also called `Anonymous functions`. There are part of the new `es6` features

> Below is the syntax

```js
const functionName = (parameters) => {
  //code block of the function
};

const sum = (num1, num2) => {
  return num1 + num2;
};

sum(1 + 3); //This will output 4
```
