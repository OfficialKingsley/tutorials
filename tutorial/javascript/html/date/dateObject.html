Javascript Date Object

Javascript dates help us to work with dates on our website. For example if we want to add a calendar to our site. A new date object can be created using the new keyword as shown below.

let myDate = new Date(); //Nothing in the parentheses

Now that you have created a new date object, you can assign methods to it like

myDate.getFullYear()

Working with specifics

You can have the date and time written with specific values in the parentheses

Using milliseconds

When writing dates, javascript uses what is known as the time stamp to get the date but displays the date in a human readable form. The time stamp is the number of milliseconds that have passed since Jan 1st 1970 and is based something we call UNIX time

Example let milliseconds = new Date(988746938423);
console.log(milliseconds)
Output 
Tue May 01 2001 20:55:38 GMT+0100 (West African Time)

Using the date string

let dateString = new Date("Jan 21 2021 05:30:00")

OR

let dateString = new Date("2021-01-03T5:30:00+01:00")
//Here, moths are not zero indexed but just note that by default they are
//The T separates the date from the time
//The + sign shows the time zone that the user is in

Date and time components

Syntax let components = new Date(year, month, date, hours, minutes, seconds, milliseconds)

Example let componentDate = new Date(2021, 00, 03)
console.log(componentDate) //Jan 03 2021 00:00:00

If only the year and month parameters are specified, Javascript will use default values for the other values

Javascript date methods

Generally, javascript date methods can be broadly classified into two which are:

  • Get methods
  • Set Methods

Get Methods

  • getFullYear()
  • getMonth()
  • getDate()
  • getDay()
  • getHours()
  • getMinutes()
  • getSeconds()
  • getMilliSeconds()

There are so many other get methods but the above are often used to get time based on the user's time zone. However, we can get the UTC time zone which is the primary standard by which the world regulates time.
Each of the main get methods has a UTC version

Set Methods

These are relatively easy. As get methods get an already pre-defined time, the set method actually sets the value of the time. For every get method there is a set method

  • setFullYear()
  • setMonth()
  • setDate()
  • setDay()
  • setHours()
  • setMinutes()
  • setSeconds()
  • setMilliSeconds()

Just like the get method, there are many more set methods

Getting the date/time in text

Normally, when we get dates, we get them in numbers but this can be resolved using the toLocaleString method

syntax currentDate.toLocaleString("language", object )
Example let currentDate = new Date();
currentDate.toLocaleString("default", {month: "long"} )
Example let currentDate = new Date();
currentDate.toLocaleString("en", {
month: "long"
weekday: "short"
timezone: ""
} );
Example let currentDate = new Date();
currentDate.toString()

Example let currentDate = new Date();
currentDate.toISOString()
result 2021-09-08T04:10:45.000Z

The above yields an ISO-86-01 formatted date time string in UTC
