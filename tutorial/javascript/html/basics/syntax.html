Javascript Basic Syntax

A program is a set of instructions that tells the computer what to do. These instructions are called statements. In javascript, statements are separated by semi-colons 

var name = "Kingsley";
var age = "16";

let course = ["English", "Physics", "Math"];

Basic Terms in JS

Strings
A string is something written in quotes e.g.
"Hello World a new string"
Numbers:
These can be written with or without decimals. In Javascript there are no special numbers like floats.
Variables
These are like containers for storing information
Operators
Expressions
Keywords:
These words are special words that are used to define specific actions in JS
