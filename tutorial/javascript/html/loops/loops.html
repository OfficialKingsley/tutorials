Loops

A loop is a statement or function that helps code to run over and over again

Examples let cars = ["Volvo", "Benz", "Toyota", "Nissan" "Innoson"]
Instead of writing: 
cars[0]
cars[1]
cars[2]
cars[3]
cars[4]

You can also type,

for (i = 0; i < cars.length; i++) { 
    text += cars[i] + " "; 
}

Types of loops

  • For
  • For in (for objects)
  • For of (for arrays)
  • While
  • Do while
  • Switch statement
