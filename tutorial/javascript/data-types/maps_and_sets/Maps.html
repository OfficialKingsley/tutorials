Maps

A map is also a list of key value pairs in the form of arrays. e.g.

let myMap = new Map([["a1", "Hello"], ["b2", "World"]])
//You'll see first the braces () then an array containing arrays of key, value pairs

console.log(myMap);
//This will output:
//
Map("a1" => "Hello", "b2" )

To add an item to the map, we call the set method

myMap.set("c3", "Goodbye")

To delete, we call the .delete(key) method.

WeakMaps

Just like maps, it has to do with key value pairs

let carWeakMap = WeakMap();


let key1 = {
    id: 1
}
let car1 = {
    make: "Honda",
    model: "Civic"
}
carWeakMap.set(key1, car1)

console.log(carWeakMap);

//WeakMap {Object {id: 1} => Object {make: "Honda", model: "Civic"}}
