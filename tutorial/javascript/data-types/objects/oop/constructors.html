Factory functions

When an object has one or more methods, it has a behavior. Creating an object with object literals may be an issue if it has methods. Also if you want to duplicate an object, you will have to re-write the code or copy and paste over and over again.
To solve this issue, we can define functions for creating objects. These functions are generally divided into two which are Factories and Constructors 

function createCircle (rad) {
    return {
        radius: rad,
        draw:function() {
            console.log("draw")
        }
    }
}
const circle = createCircle(1)
constructor 
function Circle(rad) {
    this.radius = rad, //Notice the this keyword
    this.draw = function() {
        console.log("draw")
    }
}
const anotherCircle = new Circle(1);
Every object in JS has a property called constructor which shows the function used to create the object. Even functions ini JavaScript also have constructors
