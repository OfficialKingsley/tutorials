Array like objects

First, what are array like object?
Array like objects are object representations of arrays with a length property

e.g.
Array 
let realArray = ["a", "b", "c"]
Array like 
let arrayLike = {
  0 : "a",
  1 : "b",
  2 : "c",
  length : 3 //notice the length property
}

Common examples of array like objects are the array of arguments in objects. So in essence it is an object containing the arguments where the keys are the index and the values are the actual parameters

Another example is the list of nodes we get when we use methods like document.querySelectorAll()

One major difference is that the array like objects are objects so they inherit from the Object.prototype while arrays inherit from Array.prototype. Therefore, array like objects don't have methods like: 

          .forEach()
          .push()
          .map()
          .filter()
          .slice()
          .indexOf()

Converting array like objects to arrays

Method1:

const arrayLike = {

  length : 2
}

  realArray.append(value);

Method3: Using the spread operator

let realArray = [...arrayLike]

Method4

Method5

let realArray = Object.keys(arrayLike).map((key)=>arrayLike[key])

The reduce function

This is simply a function that reduces or accumulates all the values of an array into one item in the array

For example, if you need to sum all the values of a particular array, you can use a loop or something but the reduce function is available for that

How does it work? It loops through the array and does a something based on a function defined in the code

Syntax 
arrayName.reduce(function(accumulator, current){}, startingValueForThe accumulator)
//Accumulator is the value gotten so far from the reduce function
//Current is the current value in the array that you are working with

For example, if you have an array and you want to sum all the items of the array together, we can do 

let arr = [1, 2, 3];

arr.reduce((a, c) => {
  a + c
}, 0)

//here, a is the accumulator
//c is the current (value)
//So at first, the accumulator is 0
//Since that is the starting value of the accumulator

//So according to the above code, what we have done is that
//We have set a to 
//a = a + c
//Then in the final iteration, it returns "a"

In the first iteration, 
a = 0 + 1
//now a is 1
//In the second iteration,
a = 1 + 2
//now a is 3
//In the third, which is the last because there are only 3 values in the array
a = 3 + 3;

The map method

The map method also loops through the array and performs a specific action 

array.map(function (3 possible parameters) {}, optional object parameter)

//the three possible parameters are 
//1. the particular value in the array in the iteration
//2. The index of that particular value
//3. Represents the whole array that you are looping throgh

//The optional object parameter is used if we want to use the this keyword in the map method
//This object is used so that when we type the this keyword, 
//it will refer to the optional object passed and not the global window object

example

let myArray = [1, 2, 3, 4]

myArray.map((value, index, arr) => {
  console.log(`${value} has an index of ${index} in the array ${arr}` )
})

//With the optional object,

myArray.map((value, index, arr) => {
  console.log(`${value} has an index of ${index} in the array ${arr}`, this.documentation )
}, {documentation: "this is the optional object"})
//Output: 1 has an index of 0 in the array [1, 2, 3, 4] this is the optional object

The sort method

By default, the sort method sorts the items of an array according their string unicode. It does not create a new array like other methods but it modifies the original

It sorts items in this order, 

Negative numbers
Whole numbers
Uppercase letters
Lowercase

Alphabetical String Sort

let thisArray = ["s", "t", "a", "c", "K", "o", "v", "E", "r", "f", "l", "w",
"2", "1"];
thisArray.sort((a, b)=> { return a.localCompare(b) });

//Output

["1", "2", "a", "c", "E", "f", "K", "l", "o", "r", "s", "t", "v", "W"]

The above sort will throw an error if any array item is not a string.
If you know that the array may contain items that are not strings use the safer version bellow 

thisArray.sort((a, b) => {
  return a.toString().localCompare(b)
})
