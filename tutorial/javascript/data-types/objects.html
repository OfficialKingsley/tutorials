Objects

In real life, a car is an object. A car has properties like weight and color, and methods (functions) like start and stop: Examples of a car's properties are its name, model, weight, year etc. Examples of a car's methods are start, stop, accelerate.

example
Properties car.name = Fiat;
car.model = 500;
car.weight = 850kg;
car.color = white
Methods car.start();
car.drive();
car.brake();
car.stop();

All cars (which are objects) have the same properties but different property values. The same applies for methods but these methods are performed at different times

Basic definition

Objects in JavaScript are also variables which can contain many values. These values usually come in key: value or name: value pairs.

Example var car = {
type: "",
color: "white",
makeYear: "2020",
}

In other words JavaScript objects could be defined as the containers for named values

Object Properties

The name: value pairs are called the properties. To access these property values, you need to know the name

var person = {
firstName:"John",
lastName:"Doe",
age:50,
eyeColor:"blue"
};


Property Property Values

Accessing object properties

There are two ways to access the object properties

  1. Using the dot operator
  2. Using the square brackets
with the dot var person = {
firstName:"John",
lastName:"Doe",
age:50,
eyeColor:"blue"
};
console.log(person.firstName) //This will yield "John"
With the square brackets var person = {
firstName:"John",
lastName:"Doe",
age:50,
eyeColor:"blue"
};
console.log(person["firstName"]) //This will also yield "John"

Methods

These are simply actions that can be performed on objects. They can also be simply defined as the functions of objects

To create a method, you add a property which has a value of a function

With the square brackets var person = {
firstName:"John",
lastName:"Doe",
age:50,
eyeColor:"blue"

getMe: function() {code to be executed}
};
person.getMe() //Notice the parentheses
With the square brackets var person = {
firstName:"John",
lastName:"Doe",
age:50,
eyeColor:"blue"

getFullName: function() {firstName + " " + lastName}
};
person.getFullName() //This will yield "John Doe"

Note:

  • Do Not Declare Stri ngs, Numbers, and Booleans as Objects!
  • When a JavaScript variable is declared with the keyword "new", the variable is created as an object.

var x = new String(); // Declares x as a String object
var y = new Number(); // Declares y as a Number object
var z = new Boolean(); // Declares z as a Boolean object
