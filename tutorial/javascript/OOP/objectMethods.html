TO compare strings lexicographically, (or alphabetically) you can use < or >. The letter "a" is greater than the letter "b"

You can also compare strings useing the localeCompare e.g. a.localeCompare(b)

Arrays

Array.from changes an array like object to an actual array. An array like object does not have the array methods like forEach().

[...arrayLikeObjecName] The syntax also changes an array like object to an array

Object.value(arrayLikeObjecName) does the same as the above

Objects

  • Object.freeze(objName): This freezes the object so it can't be modified. That ie its content can't be edited, You cant append other properties etc.
  • Object.seal(objName): This is similar to object.freeze but instead you can edit the content but can't add or remove from the object.
  • Object.entries(objName)
  • Object.getOwnProperties()
  • Object.defineProperty(objName, "propertyName" {value: "original", writable: "false" makes it readOnly}
