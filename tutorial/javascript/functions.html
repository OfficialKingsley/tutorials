A function is a code designed to perform a specific task.
A javascript function only executes if it has been called or invoked.
In javascript, functions can be defined using the function keyword.

function nameOfFunction () = {
Code to be executed when the function is called
}

Invoking a function

To invoke or call a function, all you need do is to write the function name with parentheses. e.g.
function myFunction() {
console.log("Hello world")
}

myFunction() //Here the function has been called

Parameters

A parameter can be defined as a variable defined with a function.

function myFunction(parameter) {
console.log(parameter)
}

When calling the function, you also have to specify the parameter 
myFunction("Kingsley") //This will log out Kingsley

You can also specify multiple parameters

function myFunction(name, age) {
console.log(name + " is" + age + " years old")
}

You can also predefine the parameter values

function myFunction(name = "Kingsley", age) {
    console.log(name + " is" + age + " years old") 
}

If you know that the user can input multiple parameters, you can use the rest operator to achieve that
