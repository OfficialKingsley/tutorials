Variables in JavaScript

A variable can simply be defined as a container for a particular information. In javascript, we can declare variables using keywords let var and const

Using the var keyword is the old way of defining variables so we use the let and const

Difference between let and const

The major difference between let and const is that when let is declared, its value can be changed but if the variable is declared with const, the value can't be changed 
let name = "Kingsley"; 
const age = 16; 

name = "Oluchi"; //This is a valid re-declaration 

age = 14; //This will result in an error because the const keyword was used

Rules for naming in JS

  1. It must start with a letter, underscore or a dollar sign but never a number
  2. It should not have (mathematical) operators in its name
  3. Naming in javascript is case sensitive therefore, liveSpan and livespan are not the same
  4. Javascript keywords cannot be used as variable names e.g. this
  5. There are no spaces in javascript names

Declaration and initialization

Declaring a variable is giving the variable a name without a value

let name;

The above is a declaration

let name = "My Name"

Now the above is an initializations

It should however be noted that when using the const keyword, you cannot have a declaration instead you can only have an initialization

Assigning Variable

A variable can be assigned after it has been declared even if the const keyword is used

let name;
const age;
name = "My Name"
age = 37 //Now it can not be reassigned since the const was used

I'm pretty sure that you've seen the equal to sign = So often but you should note that the equal to sign in JS is known as the assignment operator and not an equal to

You can make many declarations on the same line by separating them with commas

let name, age = "16", car = "Lamborghini";

Now you see that you don't have to declare all of them using the let keyword. This rule also applies for the const and var keywords

If a variable is not assigned a value, it has a value of undefined by default.
Also, if you declare a variable after it has been assigned a value, then it will still retain its old value

Operations in strings

We can perform mathematical operations and assign its value to the string e.g.

let x = 3 + 5 + 7; //Now x has a value of 15
console.log(x)

We can also add strings but instead of performing mathematical operations in concatenates the strings i.e. it puts the strings together to form a new string

let age = "1" + "6" //This won't give a value of 7 

console.log(age) //Instead it will give 16 

let name = "John" + " " + "Doe" 

console.log(name) //This produces John Doe
