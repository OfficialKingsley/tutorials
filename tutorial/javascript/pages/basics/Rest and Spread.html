The rest and spread operators

Both operators have the same sign or syntax that is the ellipses sign (...)
It is either called rest or spread depending on when it is used

If it is used when defining a function, then it is called rest

const sum = (...numbers) => {
    //code to add numbers
}

In the above it is called the rest operator since it was used in a (functionn) definition.
Also note that the variable numbers is now an array containing all the numbers that the user will enter into the function

On the other hand, it is called a spread when using it as a value for something. It is usually put infront of variables that are arrays

const arr = [1, 2, 3, 4, 5,6 7, 8]

The spread operator is used when calling a function so it can spread across all the values in an array or object or list of values
