Comments in JS

What is a comment?

A comment is a statement in a code that won't be executed. It is a text or little code usually used for reference or for other programmers to know the use of the code

In JavaScript there are two ways we write comments. We either write them as single line comments or multiline comments.
To make a single line comment we use two backslashes //

let name = "Kingsley" //This is a single line comment

However, to make a multiline comment, we put the comment in /* This */

let name = "Kingsley" /*This is a multiline comment as it can
span multiple line*/
