XML HTTP requests (xhr)

The name XMLHttpRequest has historic reasons, it is NOT restricted to fetching XML only. JSON is typically used nowadays 

const getData = () => {
  const xhr = new XMLHttpRequest();  //This xhr is an object
  xhr.open("GET", "https://myAPI/all/users");  //It prepares http requests to be made
  //This first parameter specifies the type of method you want to perform on the API
  //The second parameter is the API

  xhr.onload = () => {
    const data = JSON.parse(xhr.response)  //This refers to the response from the API
  }

  //We can also do xhr.responseType = "json";
  //const data = 
  xhr.send(); //now we send the request

  //xhr.send(JSON.stringify(data))
}
        
const sendHttpRequest = (method, url) => {
  const xhr = new XMLHttpRequest();
  xhr.open(method, url);
  xhr.responseType = "json";

}

Promises

const sendHttpRequest = (method, url) => {
  const promise = new Promise((resolve, reject) => {
    const xhr = new XMLHttpRequest();
    xhr.open(method, url);
    xhr.responseType = "json";
    xhr.onload = () => {
      resolve(xhr.response)
    }
  });

  return promise;
}

const getData = () => {

Handling errors

To handle errors, we use success codes in xhr

xhr.onload = () => {
  if (xhr.status >= 400) {
    reject(xhr.response)
  }
}
