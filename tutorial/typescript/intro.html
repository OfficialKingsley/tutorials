What is TypeScript

It is a javascript superset. This means that typescript is a language that builds on javascript. It makes writing javascript easier and adds features and advantages to it.
It should however be noted that typescript cannot run on the client side or browser and node js also can't execute typescript

Typescript is a compiler which compiles into javascript. You write code in typescript and then you get an output in javascript.
Typescript also adds types and helps you to give you extra error checking 

const button = document.querySelector("button");
const input1 = document.getElementById("input1")
const input2 = document.getElementById("input2")

function add(num1, num2) {
  return num1 + num2
}

button.addEventListener("click", function() {
  console.log(add(input1.value, input2.value))
})

In the above, you will realize that the function will concatenate because the values gotten from an input are always strings no matter the type of input it is even if it's of type number.
The error is not as a result of bad code but however this can be avoided in typescript because it is going to give errors there

intro.ts 
const button = document.querySelector("button")!;
const input1 = document.getElementById("input1")! as HTMLInputElement;
const input2 = document.getElementById("input2")! as HTMLInputElement;

//The above is called type casting

function add(num1, num2) {
  return num1 + num2
}

button.addEventListener("click", function() {
  console.log(add(input1.value, input2.value))
})

We might be able to see that the argument that can be passed into the add function can be of any type but we do not want that. We want the add function to only be able to accept number values.
We can therefore specify the type of parameters in our function definition 

const button = document.querySelector("button")!;
const input1 = document.getElementById("input1")! as HTMLInputElement;
const input2 = document.getElementById("input2")! as HTMLInputElement;

//The above is called type casting

function add(num1: number, num2: number) {
  return num1 + num2
}
//We used colons to add the type of data to be passed


button.addEventListener("click", function() {
  console.log(add(input1.value, input2.value))
  //Then we get another error here because
  //the value property from an input element is always a string
  //no matter the type of input it is
  //So we want to change the strings to numbers
})
