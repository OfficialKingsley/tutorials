In a case where you may like to have some global variables for some numbers or for some strings that's where enums come in

This is a custom type

enums {NAME1, NAME2}
        
enum Role {ADMIN, READ_ONLY, AUTHOR}
Role.ADMIN //0
Role.READ_ONLY //1
Role.AUTHOR //2

If for some reason you don't want that to start from 0, you can do this 

enum Role {ADMIN=12, READ_ONLY, AUTHOR}
//Now Role.READ_ONLY == 13
        
enum Role {ADMIN="ADMIN", READ_ONLY=1, AUTHOR=23}
