let combineValues: Function

Function types are types the describe a function regarding

variable = (...paramTypes) => return type
let combineValues: () => number

For the above code, we are telling typescript that combineValues should be a variable that takes in no parameters and returns a number from the return statement

let combineValues: (number, string) => number
//Here, you're telling TypeScript that
//combineValues variable should accept a function
//Which has two parameters
//Where the first is a number and the next is string
            
These function types are useful in call backs

You should also note that when the type of a function return is set to void, even if the function returns a value, that value will be avoided
