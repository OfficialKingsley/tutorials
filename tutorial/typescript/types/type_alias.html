You might not want to keep repeating writing the types for some code. Like for example, you might not want to keep repeating string | number so many times so what you can do is to create an alias 

//type name_of_your_customizable = type (or types);

type Combinable = number | string
type ConversionDescriptor = "as-number" | "as-string"

So you don't need to keep writing those unions and literal types
