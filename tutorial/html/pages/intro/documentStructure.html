HTML document structure

<!DOCTYPE html>
<html>
<head>
<meta name="" content="Some meta information">

<title>The title of the page</title> </head>
</head>

<body>
<h1>My First Heading 1</h1>

<p>My First Heading 1</p>

<p>My First Heading 1</p>
</body>

The part that is visible to the user is the body. It is in the body that the work is done

When designing your website make sure you do the following:

  • Always declare your doctype
  • Always put your html element as it describes the document
  • Be sure to put your head tag which contains vital information about your website
  • The title must be nested inside the head element


The html Element

This defines the document of the webpage and is compulsory in all html documents

The head Element

The head element gives information about the website (especially) to browsers. In the head element we can also find some other tags and such as:

  • Meta
  • Style
  • Link
  • Script
  • Base
  • Noscript
  • Title: Which specifies the title of the page
