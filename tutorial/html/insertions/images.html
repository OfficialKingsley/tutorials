In this course we are going to be learning about images in HTML

To add an image to a page, we use the image element

The alt attribute is used to show a text if the image is not displayed and it is important to write good alt text for your images

It should also be noted that there is a semantic difference between including an empty alt attribute and excluding it altogether. An empty alt attribute indicates that the image is not a key part of the content and the image is just an additive image that is not necessary to understand the rest and thus may be omitted from rendering. However, the lack of an alt attribute indicates that the image is a key part of the content and that there simply is no textual equivalent available for rendering.

You can change the width or height of your image using the width and height attributes. The width can only be in pixels(px) or percentage(%) while the height can only be in pixels

<img src:"Image URL" alt="Statement if image does not load" width="x% or ypx" height="y px" >
