SASS tutorial

Sass is compiled to CSS

SASS variables

$my-color: #005dff;
body {
    background-color: $my-color;
}

In these, it appears as

body {
    background-color: #005dff;
}

SASS maps

$colors: (
    primary: #005dff;
    accent: #fff6bb;
)
/* syntax: 
selector {
    property: map-get(variable-name, property in map)
}*/
body {
    background-color: map-get($colors, primary);
}

CSS output

Functions

@function function-name($parameter) {
    @return do-something-with-the-parameter
}
                
@function color($color-name) {
    @return map-get($colors, $color-name)
}
body {
    #bg {
        background-color: color(primary); 
    }
}
