Node modules

Node comes with some core modules like path, fs and http.
You can create your own modules and you can install modules via npm

Node comes with what is called a repl which stands for read, eval, print, loop which helps us write js

A module is a js file that has closely related functions together

To export a function from a module, we can do

const sum(one, two) => return one + two; 
//Function we wanna export

module.exports = sum; 
//Now you've exported the function
        
const tutorial = require("./tutorial") 
//Now we have imported the sum function

console.log(tutorial); 
//Output [function: sum]

Now what if we want to export multiple items

const sum(one, two) => return one + two; //Function we wanna export
module.exports.sum = sum; //Now you've exported the function
        
const tutorial = require("./tutorial") 
//Now we have imported the sum function

console.log(tutorial.sum); 
//Output [function: sum]

console.log(tutorial)
//An object

This can further be simplified as

const sum(one, two) => return one + two; //Function we wanna export
const PI = 3.14;
module.exports = {sum: sum, PI : PI}; //Now you've exported the function
