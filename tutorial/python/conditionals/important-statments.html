The and statement is used to check if both condition in a conditional statement are both true 

if 1<2 and 4>3:
    print("Both are true")

#Output: "Both are true"
if 1<2 and 4<3:
    print("Both are true")

#The above code will not run

The or statement is used when one of the conditions are true i.e. either the first or the second are true
