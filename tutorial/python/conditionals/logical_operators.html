The logical operators

  • and
  • or
  • not

The and statement is used to check if both condition in a conditional statement are both true 

if 1<2 and 4>3:
    print("Both are true")

#Output: "Both are true"
if 1<2 and 4<3:
    print("Both are true")

#The above code will not run

The or statement is used when one of the conditions are true i.e. either the first or the second are true or both are true however if both are false, the code won't run 

has_good_credit = True
has_criminal_record = False

if has_good_credit and not has_criminal_record:
  print("Eligible for loan")
