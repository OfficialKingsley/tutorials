You can join lists together using the extendlist

list1 = [1, 2, 3, 4, 5]
fruits = ["banana", "apples", "mango", "oranges"]

list1.extend(fruits)
print(list1)

To append (or add to the back) of a list

fruits.append("Cherry")

You can also insert it at a certain index

fruits.insert(1, "cherry")
                
fruits.remove("banana");
                
fruits.clear()

To get the index number of an element

fruits.index("mango")

To know the number of times something appears in a list, we type

fruits.count("mango")
