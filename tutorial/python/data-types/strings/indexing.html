The index of the item in a string refers to the position of the item in the string and indexing generally refers to getting an item based on its index.
In python, or generally in programming, indexes start from 0
That is, the first item has a position of 0, the second will have an index of 1, the third has an index of 2 and so on
You can do the indexing process using square brackets 

name = "Kingsley"
print(name[0]) #K
print(name[1]) #i
print(name[3]) #g

If I try to get something that its index is not possible, I get an index error 

name = "Kingsley"

print(name[10]) #IndexError

Negative indexing

print(name[-1]) #y
print(name[-2]) #e
                
print(name[1:])
print(name[1:5])
print(name[:5])
print(name[:]) #This will print the whole string
