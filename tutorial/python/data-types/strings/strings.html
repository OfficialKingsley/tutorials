Strings are basically anything within quotes

print("Hi, How are you")
#Output: Hi, How are you
                
print("Hi, \nHow are you")
#Output:
Hi, 
How are you

Notice that the How are you was placed on a new line and that was because of the special character \n. Special characters basically start with a backslash

You should note that when using single quotes for your strings, you should not have single quotes in there 

print('He's good')

The above will give an error because python will think that you have ended the string after He and then treat s and good as variables or something like that
To avoid this, we can do the following 

print("He's good")
#Output: He's good
#We used double quotes for the strings

print('He\'s good')
#Output: He's good
#We escaped that ' with the backslash
