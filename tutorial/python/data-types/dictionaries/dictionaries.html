These are like objects in javascript . They store data in key-value pairs 

my_dict = {
    "name": "Tim",
    "nationality": "African",
    "age": 23,
    "qualification": "College"
}

To get a particular value of dictionary, you need to get the value by using the key 

print(my_dict["name"]) #Tim

If you have two values with the same key, the most recent one is what is going to be taken 

my_dict = {
    "name": "Tim",
    "name": "John",
    "nationality": "African",
    "age": 23,
    "qualification": "College",
    "is_tall": True,
    "friends": ["Peter", "paul", "Precious"]
}

print(my_dict["name"]) #John
print(len(my_dict))
print(type(my_dict)) #<class dict>
            
customer = {
  "name": "John Smith",
  "age": 30,
  "is_verified": True
}
print(customer["birthdate"])
#KeyError "birthdate"

print(customer.get("birthdate")) #None
print(customer.get("birthdate", "Jan 1 1980")) #Jan 1 1980
#That output will be the default 
#if the item with that key does not exist

customer["birthday"] = "Jan 3rd" #Assigning a new key-value pair
