You might want to work with some external files like a spreadsheet or something

For example, if we create a new file in the same directory as our python file, and let's name it countries.txt 

Ghana
Mexico
France
Nigeria

To read a file, we call the open function and pass in two parameters. The first parameter is the location of the file and the second parameter is what we want to do to the file like to read, write, append etc. 

my_file = open("file", "r")

my_file.close()

#In the above, the second parameter is r which means read.
Other (second) parameters include
"w" - To write (or edit) the file
"r+" - To read and write
"a" - To append to the file ""

Notice that we also closed the file after we were done with what we were doing. It is always good to close the file after you are done with it

To check whether the file is readable, you can use the .readable() method

python file 
countries_file = open("file", "r")

print(countries_file.readable()) #returns True or False

countries_file.close()

To reading the first line

countries_file = open("file", "r")

print(countries_file.readline()) 
#returns the first line

countries_file.close()

Note that we can call the .readline() method many times, everytime we call it, it returns the next line
For example if we call the readline() method 3 times, it is going to return the first three lines and for each time we call it, it returns just one line 

countries_file = open("file", "r")

print(countries_file.readline()) #returns the first line
print(countries_file.readline()) #returns the second line
print(countries_file.readline()) #returns the third line


countries_file.close()

However, we can simply get all the lines of the file by using the .readlines() method.
Notice I wrote readlines and not readline
The .readlines() method returns all the lines in the form of an array. Therefore, we can index out the line that we want 

countries_file = open("file", "r")

print(countries_file.readlines())
#Returns an array of all lines

print(countries_file.readlines()[0])
#prints first line

print(countries_file.readlines()[2])
#prints third line

for line in countries_file.readlines():
    print("line")

countries_file.close()
