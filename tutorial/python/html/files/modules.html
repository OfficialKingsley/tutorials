A module is a folder that contains files which contain similar functions and other properties.
These functions can be exported
You can create your own modules.
Imagine a supermarket, each section in the supermarket is like a module. In each section we could see subsections for particular companies which are like the files

new.py 
def say_hi():
    print("Hi")
                    
import new
#import new.py
#you've imported the whole module like an object

new.say_hi() #Hi
                    
from new import say_hi

say_hi()
#Hi

The advantage of modules is that sometimes you don't have to write all the code for some projects. It might be that someone out there has done a similar project and might have written a function you want to write. You can simply install the person's module and import it into your file

A lot of modules are hosted online and so you can install them using the python pip (which is like npm for node)
