Like I said in the function tutorials, parameters are the variables of the function

  • positional arguments
  • keyword arguments

Positional arguments are the arguments passed in which their position matters 

def greet_user(first_name, last_name):
  print(f"Hello {first_name} {last_name}")
  print("Welcome aboard")

print("Start")
greet_user("John", "Smith")

In the above it can be seen that if we change the position of the arguments, Smith becomes the first_name and John becomes the last_name. That means that the position of these arguments actually matter

Keyword arguments

In these the position do not matter

def greet_user(first_name, last_name):
  print(f"Hello {first_name} {last_name}")
  print("Welcome aboard")

print("Start")
greet_user('Smith', first_name="John")

As you can see, we explicitly set the first name to John. The argument where you assign the parameter to the argument is called a keyword argument.
It is however that advised that you mostly use positional arguments but there are some cases where you may need to pass in some keyword arguments
When you are using functions that use numerical values you might want to use keyword arguments to increase readability of the code

Make sure positional arguments always come before the keyword arguments
