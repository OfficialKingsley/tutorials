Lists as the name implies is a list of items. A list is usually placed in square brackets []
Each list item is separated by a comma 

countries = ["United Kingdom", "Ghana", "Nigeria", "Australia"]
                
print(countries[0]) #United Kingdom
You can also chain indexing 
print(countries[2][0]) #N
                
print(countries[2:]) This gets the item starting from index 2 till the end
print(countries[2:4])) This gets from item index of 2 to item index of (4-1) 3

Negative indexing

You can also index with negative numbers. The index of the last item in the list is (-1) the second to the last will have an index of -2 

print(countries[-1])
                
countries = ["United Kingdom", "Ghana", "Nigeria", "Australia"]
countries[0] = "United States"
print(countries)
#["United States", "Ghana", "Nigeria", "Australia"]
                
countries = ["United Kingdom", "Ghana", "Nigeria", "Australia"]
countries[0] = "United States"
print(countries)
#["United States", "Ghana", "Nigeria", "Australia"]
                
print(len(countries)) #4

YOu can also have multiple data-types in a list

countries = ["UK", 2, True, {"name": "Kingsley"}]
                
countries = list(("UK", 2, True, {"name": "Kingsley"}))

print(countries)
["UK", 2, True, {"name": "Kingsley"}]
