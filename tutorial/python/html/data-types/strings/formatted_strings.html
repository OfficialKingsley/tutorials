Formatted strings

These make strings easier to visualize and also prevents the excess concatenation of strings 

first = "John"
last = "Smith"
message = first + " [" + last + "] is a coder"

The above can be simplified to formatted strings. Formatted strings always begin with an f just before the strings. This is similar to template strings in javascript.
Anytime we want to add some code (like a variable), we enclose it in curly braces {} 

first = "John"
msg = f'{first} [{last}] is a coder'
        
msg = f'''{first} can see that we can also use
formatted strings with multiple line strings'''
