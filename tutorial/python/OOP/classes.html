A class can be defined as a constructor of objects

class NewClass:
    //Add attributes and methods to the object

To create an object from the class, we initialize it

p1 = NewClass()

To assign attributes to the class,

class Line:
    #attribute_name = attribute_value
    x = 1
    y = 4

These attributes can be gotten from the object using the dot operator

myLine = Line()

print(myLine.x) #1
print(myLine.y) #4

init function

This allows us to initialize the class and allow the class to accept parameters to create the object.
When creating the __init__ function in a class, it has a compulsory self parameter that must be added. After that, still in the definition of the __init__ if we want the class to accept any more parameters, we pass it there 

class Person:
    def __init__(self):
        self.x = 1

p1 = Person()
print(p1.x) #1
                
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

p1 = Person("Kingsley", 17) #You have to now pass in the name and age parameters
print(p1.name) #Kingsley

If you've worked with Javascript, that self statement is like the this statement in here. You do not control it.
Also you will notice that the __init__ function is similar to the constructor function in Javascript classes

pass statement

Incase you define a class and you're not sure of the attributes or methods yet or you want to keep that till another time, you can use the pass statement to tell python to overlook that class definition 

class Person:
    #Nothing here will generate an error when ran

class Person:
    pass
    #Now this code will run without an error
