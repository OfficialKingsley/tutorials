Functions are block of codes that perform certain tasks

def function_name(params):
    block of code here
                
def greetings_function():
    print("Welcome user")

#you have to call the function to make it run
greetings_function()

You can add parameters to the function to make the user input variables and we can do things with these 

def greetings_function(name):
    print("Welcome user")

greetings_function()
#The above will give an error because we passed in a parameter in the function definition
greetings_function("John")

You can also pass in more than one parameter

def greetings_function(name, age):
    print("Welcome" , name, "you are", age, "years old")

greetings_function("Kingsley", 27)

If you want the function to accept multiple variables or parameters, you can add the asterisk operator. After doing that, the parameters will be a list 

def greetings_function(*names):
    print("Welcome " + names[1])

greetings_function("John", "Tim", "Tom")
                
def greetings_function(name, age):
    print(name, "is", age)

greetings_function(name="Kingsley", age = 27)

Difference between parameters and arguments

Parameters are the variables created during the function definition while the argument is the object passed when the function is being called so take note
