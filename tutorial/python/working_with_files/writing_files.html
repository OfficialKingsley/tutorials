const countries_file = open("./countries.txt", "w")

countries_file.write("This is some text that will override whatever is already written in the file")

Notice that the second parameter that we passed into the open function was "w" which means to write.
Note also that the write method overrides what ever is already in the file and whatever that was there before will be replaced by this new text

However, if the file specified does not exist, it will create the file and write to it 

const country_file = open("./country.txt", "w")

country_file.write("This is some text that will added to the newly created file since it did not exist before")
