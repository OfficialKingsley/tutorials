To use a templating engine like pug, you have to load the view engine and specify the folder where your views or templates will be kept 

app.set("views", path.join(__dirname, "views"))
app.set("view engine", pug)            

//New route
app.get("/", function(req, res)=> {
    res.render("index")
})

//We are rendering the index template when
//A request is made to the home or "/" route

It should be noted that the pug engine uses indentation to work and not normal html tags

index.pug
