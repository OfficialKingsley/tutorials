In pug, we can accept variables from the express app.
To pass in the variables into pug, when we render a template, we can pass in an optional object (parameter) and in there we pass in the variables we want to use 

const app = express()

app.set("views", path.join(__dirname, "views/"))
app.set("view engine", "pug")

//Adding a route
app.get("/", function (req, res) {
    res.render("index", {
        //This object contains the variables
        title: "Memories",
        description: "This is a wonderful description of memories"
    })
})

Now to use those variables, all you have to do is to put it in curly braces with a pound sign in front of the curly braces as so:
#{variable}
