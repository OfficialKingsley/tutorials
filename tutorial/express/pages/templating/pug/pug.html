doctype html
html(lang="en")
    head
        meta(charset="UTF-8")
        meta(http-equiv="X-UA-Compatible", content="IE=edge")
        meta(name="viewport", content="width=device-width, initial-scale=1.0")
        title Document
    body
        h1 This is a heading 1
            span
        p I like the way this paragraph is

If you look at the above, you will see that there are not tags. If you want an element inside another, you have to just tab into it.
In the above, the h1 and p are children of the body tag since they are tabbed into it. The span tag is also tabbed into the h1 therefore, the span is a child of the h1 tag

Also, you can see in the code above that attributes are added by putting them in parentheses and putting them beside the element

You can add a class to an element with the dot operator

h1.title
p.description

input.btn.btn-primary

It can also be seen that multiple classes can be added by putting many dots
