Some of the values of the res attribute include:

  • res.send()
    This is actually used to send some mere data
  • res.status(200)
    This is going to send a status code (in this case 200)
  • res.json()
    This is going to send back json data
  • res.render()
    This can be used to render a template from a template engine or something
  • res.redirect(link)
    This will actually redirect you to the link specified in the parentheses
